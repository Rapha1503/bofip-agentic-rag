Pipeline

Le projet met en avant une approche RAG fiscalement prudente : retrouver des preuves, les citer, et reconnaître les limites lorsque la couverture est insuffisante.

1

Parsing

Extraction XML/HTML du BOFiP vers des documents structurés et traçables.

2

Chunking

Fenêtrage par sections pour conserver le contexte juridique local.

3

Retrieval

BM25 multi-vues, embeddings E5-large, fusion RRF pondérée par confiance.

4

Sélection

Recherche locale dans les documents retenus et diversité des passages.

5

Génération

Prompt contraint, citations BOFiP et statut supported/partial/insufficient evidence.

Local

Clone, place les artefacts ou active le téléchargement automatique, ajoute une clé API, puis lance Streamlit.

Hosted

Le runtime est préparé pour Hugging Face Spaces via Docker, avec reranker désactivé par défaut sur CPU gratuit.

Security

BYOK explicite : la clé n'est pas commitée ni loggée ; elle transite par le serveur qui exécute la démo.
